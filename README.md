## About Me

<img align="right" src="https://raw.githubusercontent.com/AbdulrhmanBadwy/AbdulrhmanBadwy/main/assets/coding.gif" width="320" />

I build mobile products with **Flutter** and back them with **Laravel** APIs designed around clean architecture and clear boundaries between layers. My focus is on writing code that survives contact with a growing codebase — testable, modular, and easy for the next engineer to reason about.

I care about the details that separate a working app from a *good* one: state management that doesn't fight the framework, API contracts that don't leak implementation details, and databases that are modeled instead of improvised.

Outside of shipping features, I spend time reading source code from projects I admire, refining how I structure REST APIs, and looking for the smallest correct abstraction rather than the cleverest one.

**Currently based in** Egypt 🇪🇬 · **Studying** Computer Science
